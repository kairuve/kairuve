# GROWTH — Estrategia de Crecimiento

> Mantenido por Perplexity + Rodhan. Claude Code NO modifica este archivo salvo instrucción explícita.
> Última actualización: 2026-06-30 | 📝 Provisional

## Estado comercial actual
- Fase: Hardware planificación — 0 unidades construidas
- Modelo: Open-source DIY (CAD + firmware abiertos)
- Stack: Jetson Nano + ROS Melodic + Arduino Mega + steppers NEMA 17/23

## Usuario objetivo
- **Makers**: "tercer brazo inteligente" para soldar, ensamblar, reparar
- **Investigadores/educación**: plataforma pick-and-place open-source
- **Casos uso**: sorting objetos, assembly asistido, servicio café (demo)

## Top 3 canales de adquisición
1. **YouTube**: "DIY 4-DOF robot arm with vision (Jetson Nano + ROS)" — tutorial completo
2. **Hackster.io**: proyecto destacado con BOM detallado (~$200 USD total)
3. **Reddit r/robotics**: demo pick-and-place con visión RGB-D

## Diferenciador clave
- Hereda stack IA Makinor: Vosk STT + OpenCV (ecosistema coherente)
- AccelStepper: control preciso motores (no servos imprecisos)
- Open-source completo: FreeCAD CAD + firmware + ROS packages

## Hitos de crecimiento
- [ ] Primer prototipo funcional (4 DOF movimiento + visión RGB-D)
- [ ] Demo pick-and-place exitoso (objeto detectado→agarrado→colocado)
- [ ] Repo GitHub público con BOM + instrucciones
- [ ] Video YouTube >5k views
- [ ] Primera réplica por maker externo
