# firmware/arduino — Firmware Arduino Mega

## Descripción
Sketch para Arduino Mega encargado del control de motores stepper vía
señales STEP/DIR hacia drivers TMC2209, recibiendo comandos desde
Jetson Nano por puerto serial.

## Requisitos
- Arduino IDE 2.x
- Librería [AccelStepper](https://www.airspayce.com/mikem/arduino/AccelStepper/)

## Estado
Firmware principal pendiente de DEC-002.
