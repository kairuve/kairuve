# Kairuvë 🤖

> *Del quenya: "instrumento que fluye/alcanza"*

Sistema de brazos robóticos asistentes de taller. Open-source, económico y reproducible.
Diseñado para makers, constructores de gadgets y técnicos en reparación de equipos.

## ¿Qué hace?
- Sostiene piezas en posición mientras tienes ambas manos ocupadas
- Recibe y guarda herramientas por comando de voz
- Detecta objetos con visión artificial (cámara RGB-D)

## Stack
NVIDIA Jetson Nano 4GB · ROS Melodic · MoveIt · Vosk · OpenCV · Orbbec Astra Pro Plus
